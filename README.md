# Vencedores de Sangrar

Una novela de misterio ambientada en Perú que explora temas de poder, corrupción y justicia.

## Acerca del Proyecto

Este repositorio contiene los capítulos de la novela "Vencedores de Sangrar" publicada como un sitio web estático usando GitHub Pages y Jekyll.

## Estructura del Proyecto

- `_layouts/`: Contiene las plantillas HTML para los capítulos
- `Capítulo XX - Título.md`: Archivos de los capítulos individuales
- `index.md`: Página principal con el índice de capítulos
- `_config.yml`: Configuración de Jekyll para GitHub Pages

## Cómo Leer

Visita la [página web del libro](https://carlosvidal.github.io/Vencedores-de-Sangrar/) para leer la novela completa.

## Desarrollo Local

Si deseas ejecutar este sitio localmente:

1. Instala [Jekyll](https://jekyllrb.com/docs/installation/)
2. Clona este repositorio
3. Ejecuta `bundle install` para instalar dependencias
4. Ejecuta `bundle exec jekyll serve` para iniciar el servidor local
5. Visita `http://localhost:4000/Vencedores-de-Sangrar/` en tu navegador
